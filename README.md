# Matrix Rust Components Swift

Unseal's Swift Package wrapper for the Matrix Rust SDK.

This repository is intended to hold generated iOS release artifacts from
`unseal-network/matrix-rust-sdk`:

- `Package.swift`
- generated Swift bindings
- `MatrixSDKFFI.xcframework`

The iOS app should depend on tagged versions of this package instead of a local
`../matrix-rust-sdk` path.
